# Before Using Agoric Software

Before working with the Agoric CLI, Zoe, and other Agoric tools and software, you need to install the following.

Step Action Explanation
1
 Install Node.js, version 12.16.1 or higher
2
 Install Yarn 1 Yarn is a package manager for your code and lets developers share code with others. Code is shared via a package that contains all shared code and a package.json file describing the package. The link takes you to the Yarn install page, where you first select what operating system you want to install on. Your selection changes the page's content to give install instructions for that OS and links to the needed downloads.
3
 Open a shell. The rest of this table's Actions take place in this shell. A terminal on Macs; see Finder>Applications>Utilities>terminal
4
 Run git clone https://github.com/Agoric/agoric-sdk Get the latest Agoric SDK from the Agoric GitHub repository. It goes into the (created, if necessary) agoric-sdk sub-directory of your home directory.
5
 Run cd ~/agoric-sdk Change to the agoric-sdk subdirectory in your home directory.
6
 Run yarn install Install NPM dependencies.
7
 Run yarn build Build sources that need compiling. Note: build is not a standard yarn parameter, but one installed with the Agoric SDK.
8
 Install the Agoric CLI by running: yarn link-cli <agoric-cli's install directory> Select a directory to install the Agoric CLI in. For example, yarn link-cli ~/bin/agoric The install directory need not already exist. In Agoric documentation, we use ~/bin/agoric as the CLI install directory.
9
 If not already there, add the directory you installed the Agoric CLI in to your $PATH. Run echo $PATH to see $PATH's current value.
