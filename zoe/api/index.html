# Zoe

Zoe v0.5.0. Last updated 4/20/2020.

The Zoe framework provides a way to write smart contracts without having to worry about offer safety. To use Zoe, we put things in terms of "offers". An offer proposal is a statement about what you want and what you're willing to offer. It turns out, many smart contracts (apart from gifts and one-way payments) involve an exchange of digital assets that can be put in terms of offer proposals.

Start creating your own contract or build on any of our existing contracts. Explore our pre-built contracts here.

The Zoe API is divided into the following sections:

  • Zoe - This section lists the methods for deploying and working with smart contracts.
  • Zoe Contract Facet - The Zoe methods provided in this section can be called synchronously by contract code. A contract can use the Zoe Contract Facet (zcf) to do things like reallocate among offers or complete an offer.
  • ZoeHelpers for writing contracts - ZoeHelpers are functions that extract common contract code and patterns into reusable helpers.
  • Records in Zoe - Throughout the documentation you will see Javascript objects that are specific to Zoe. This section provides further insight into these objects.
